HTML CLASS - Tags para formatação de texto

Negrito

Palavras podem ser destacadas de pela tag b ou, em alternativa com a tag strong

Itálico

Palavras podem ser destacadas em itálico pela tag i ou, em alternativa com a tag em

Sublinhado

Palavras podem ser destacadas utilizando o sublinhado de pela tag u

Reduzir importância

Ao contrário do b e do strong podemos retirar importância a algumas palavras utilizando a tag small

Texto excluído visível

Podemos mostrar texto como de tivesse riscado utilizando a tag del

Elevar texto

Podemos elevar texto utilizando a tag sup

Exemplo: 52

Podemos também fazer o contrárior texto utilizando a tag sub

Marcar texto

Podemos marcar texto com a tag mark
